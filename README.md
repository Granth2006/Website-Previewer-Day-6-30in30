# 📸 Website Previewer (WebSnap)

### A modern web app to preview sites, capture device-specific screenshots, and generate full-page PDFs.

<br/>

[![🚀 Live Demo](https://img.shields.io/badge/🚀%20Live%20Demo-Website%20Previewer-6366f1?style=for-the-badge)](https://website-previewer-30in30.vercel.app/)

[![GitHub](https://img.shields.io/badge/⭐%20GitHub-Granth2006-24292e?style=for-the-badge&logo=github)](https://github.com/Granth2006)

## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>📸 Device Screenshots</h3>
      Capture screenshots of any website in Desktop, Tablet, or Mobile views instantly.
    </td>
    <td width="50%">
      <h3>📄 Full Page PDFs</h3>
      Convert entire web pages into clean, downloadable A4 PDF documents.
    </td>
  </tr>
  <tr>
    <td>
      <h3>👀 Live Preview</h3>
      Features an iframe-based live preview panel that scales dynamically based on the selected device view mode.
    </td>
    <td>
      <h3>🎨 Dark Mode UI</h3>
      A clean, modern user interface with built-in dark mode and smooth animations.
    </td>
  </tr>
</table>

---

## 🔐 Privacy First

> **We Don't Track You.**
> While the screenshots and PDFs are generated on a backend service (due to browser cross-origin limits), the generated files are never permanently stored. They exist temporarily in memory just long enough to be delivered to your device, ensuring maximum privacy.
